How It Works

As blockchains grow, so does the MEV problem

When you create an order in DeFi, you generate MEV. This MEV is worth hundreds of millions on Ethereum alone and growing every day.

Miners and predatory bots take 95% of the MEV which rightfully belongs to the users, protocols, and smart contracts that generate it. This is leading the Ethereum network down a perilous path towards centralization.

And make no mistake, MEV isn't going anywhere. Any MEV that can be extracted, will be extracted.

That's why we've created a better solution. For the network and you.
